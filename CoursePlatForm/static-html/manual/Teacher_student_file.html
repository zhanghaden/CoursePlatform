我的创意

    很多报警系统还配有与控制器相连的内置无线电接收器和便携式钥匙链无线电发送器。在下文中，我们将了解这些组件在保安装置中所起的作用.多数汽车报警系统都配有可以挂在钥匙链上的无线电发送器。通过这种装置，可以向控制器发送指令，对报警系统进行遥控。汽车报警系统上的无线电发射器与无线电遥控玩具的工作原理相同。它通过无线电波脉冲调制发送特定的信号。钥匙链发送器最主要的作用就是开关报警系统。您离开车辆关上车门后，可按下按钮启动报警系统；您返回车辆时，同样也只需要轻轻按一下就能将其关闭。在多数系统中，启动或解除车辆报警系统时，控制器会发出指令，闪烁车灯，鸣响喇叭。这样是便于让您及周围的人知道报警系统已经启动。

知识点分析

对于车辆警报器和各类开关的分析

    计算机控制单元，也是报警系统的“大脑”，可以对周围的情况进行监控并发出警报 许多先进的报警系统的“大脑”实际上是一台小型电脑。当传感器察觉到异常情况时，“大脑”就会合上开关，激活报警装置（即喇叭、车前灯或警报器）。不同的安全系统使用不同的传感器，而且传感器接入控制器的方式也不同。

课程总结
3
课程的总结和分析

    经过近一个月的努力，我最终完成了小车的设计与制作，小车也基本达到了预期的要求与功能。说实话，对于我这样一个开始时几乎没有接触任何专业知识的人来说独立制作出这种小车是很困难的。在确定了制作目标后，我便不断地在网上和图书馆搜集各种相关的资料，拼命吸取其中的知识。有些零件模块起初不知道怎么用问别人还被人取笑是菜鸟......一切都是从无到有积累起来的。通过这次竞赛，我学到了很多书本上没有的知识，也锻炼了自己独立思考和动手的能力，也培养了我绝不向困难屈服的精神，总之受益匪浅。这次竞赛还有一些小小的遗憾，例如还没有来得及学习有关protel制图和proteus仿真软件的知识，因此几个重要模块的电路板不是我亲自设计制作的。单片机的软件也没有来得及优化到位，元件布局也略显粗糙...... 总之，由于个人水平有限，作品中可能会存在诸多问题及可提高的地方，在此诚挚地欢迎老师和同学们的批评和指正。
